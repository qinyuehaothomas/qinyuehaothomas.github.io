How to use üñåÔ∏èüé®

A simple webapp deployed on Render ( damn slow ),

  1. User press the fat button
  2. Start recording, while recording, store the audio as float32array
  3. When stop is pressed, send POST request to server with the array
  4. The server reply a emotion json obj
    *It is the confidence of prediction, not the "percentage" of each emotion
  5. a function turns the emotion into different shapes and colours.
  6. Record again, and the function will add the results together and normalise, and generate again
  7. Right click the image to download!

How to run the code?

  1. clone this repo
  2. open cmd in this directory, pip install requirements.txt
  3. in main directory, python app.py or flask run --cert=adhoc
  4. open the localhost and have funüé®üé® *Notice, need a adhoc cert to run on localhost, cuz microphone access requires HTTPS connections.
