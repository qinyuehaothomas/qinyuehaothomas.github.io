StaticVideoGenerator

If you pause the video, you see nothing, but if you play the video you see something Inspired by this video Floyd-Steinburg Transformation written by ChatGPT

Result (Click to watch)

How?

It use persistence of vision (POV) effect (I guess)

  1. Original(last frame) is just random static(black and white)
  2. turn image you want to display to black and white This is the "mask"
  3. flip the color of Orginal pixel for black pixel in mask,keep for white in mask

How to Use

  1. run code, select video you want to convert, wait, done
  2. output will be saved in same folder as input.

Process
